Desarrollo local (con finch):

3. cd frontend && npm run dev

Desarrollo local (con docker):
1. docker run -d --name dynamodb-local -p 8000:8000 amazon/dynamodb-local
2. cd backend && ./mvnw spring-boot:run "-Dspring-boot.run.profiles=local"
3. cd frontend && npm run dev

Parar DynamoDB Local:
finch: finch stop dynamodb-local && finch rm dynamodb-local
